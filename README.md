## Features

* separation of layout and code through twig-templates;
* similarity of MVC structure;
* .ENV via dotenv;
* meta tags for previews in social networks;
* something else;

## Requirements
* PHP 7.1;
* Apache 2.6;

## Installation

run

```bash
sh install.sh
```

Next, make changes to .env

Directory: ``` controller\public_html ```.
